# ML/AI project - Movie Genre Classification from Descriptions
This project predicts the genres of movies based on their titles and descriptions. The movie dataset was scraped from ČSFD.cz.
## Installation guide
1. Clone the repository:
```
git clone https://github.com/maru-bor/ai_project.git
cd ai_project
```

2. Create and activate a virtual environment:
- Unix / macOS:
```
python3 -m venv .venv
source .venv/bin/activate
```
- Windows (PowerShell):
```
py -3 -m venv .venv
.venv\Scripts\Activate.ps1
```
- Windows (cmd):
```
py -3 -m venv .venv
.venv\Scripts\activate
```

3. Run the project:
```
python main.py
```
